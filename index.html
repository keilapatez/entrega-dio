    • Tipos

  • Servidor Web: Entrega páginas da web aos navegadores (ex: Apache, Nginx).
  • servidor de Aplicação: Executa programas e lida com a lógica de negócios de aplicativos.
  • Servidor de Banco de Dados: Armazena e gerencia dados (ex: MySQL, PostgreSQL).
  • Servidor de Arquivos: Armazena e compartilha arquivos na rede
  • Servidor de E-mail: Envia, recebe e armazena e-mails (ex: Microsoft Exchange).
  • Servidor DNS: Converte nomes de domínio em endereços IP.
  • Servidor Proxy: Age como intermediário entre o usuário e outros servidores, ajudando em segurança e controle de acesso.

    • Hospedagem

    Hospedagem de site é o serviço que armazena um site em servidores, permitindo que ele esteja disponível na internet para que qualquer pessoa possa acessá-lo.

  • Hospedagem Compartilhada: Vários sites dividem o mesmo servidor, sendo uma opção econômica.
  • Hospedagem VPS (Servidor Virtual Privado): Um servidor é dividido em partes, oferecendo mais controle e recursos para cada site.
  • Hospedagem Dedicada: Um servidor inteiro é dedicado a um único site, oferecendo alto desempenho e controle total.
  • Hospedagem em Nuvem: Vários servidores trabalham juntos para hospedar sites, proporcionando escalabilidade e maior disponibilidade.
  • Hospedagem Gerenciada: O provedor cuida de toda a administração do servidor, ideal para quem quer simplicidade
  • Hospedagem WordPress: Otimizada para sites em WordPress, com recursos específicos para essa plataforma

    • Linguagens server-side

    Linguagem server-side(lado do servidor) refere-se a linguagens de programação usadas para processar dados no servidor antes de enviar uma resposta ao cliente. Isso significa que o código é executado no servidor, e o resultado é enviado ao navegador, geralmente como HTML. Exemplos incluem PHP, Python, Ruby e Node.js. Essas linguagens são usadas para criar funções como login, interação com banco de dados e processamento de formulários.

    Linguagem executada no servidores =php/js/java/c#/ruby.

    Linguagens client-side

    Linguagem client-side(lado do cliente) refere-se a linguagens de programação que são executadas diretamente no navegador do usuário. Isso significa que o código é processado no dispositivo do usuário, sem precisar ir ao servidor. O objetivo é criar interatividade e dinamismo nas páginas web. Exemplos incluem HTML, CSS e JavaScript. Essas linguagens são usadas para modificar o conteúdo da página, validar formulários e criar animações, por exemplo.

    Linguagem executada no client =js

    Porque HTML não é linguagem de programação

    O HTML não é considerado uma linguagem de programação porque ele não executa lógicas, como condições, loops ou cálculos. Em vez disso, o HTML é uma linguagem de marcação usada para estruturar e organizar o conteúdo de uma página web definindo elementos como textos, imagens e links. Ele apenas descreve como o conteúdo deve ser exibido no navegador, mas não "diz" ao computador o que fazer, como as linguagens de programação fazem.
